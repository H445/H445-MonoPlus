MonoPlus
www.crystalvortex.ca


MonoPlus is a semi-clone of the Korg Monotron. Extended from a youtube tutorial from Rory Walsh.
Added features: Reverse Saw LFO / ADSR Envelope

------------------------
Install Instructions:

- VST2: Copy the .dll contained in the package to your corresponding VST2 folder. 

- VST3: Copy the .vst3 contained in the package to your corresponding VST3 folder. 

- Copy the .csd file contained in this package into each of the above corresponding folders you chose to utilize.

- All done!
